# NEMESIS REPOSITORY


Learn, have fun and enjoy.

add this to your /etc/pacman.conf with npacman or the ATT.

```
[nemesis_repo]
SigLevel = Never
Server = https://erikdubois.github.io/$repo/$arch
```
